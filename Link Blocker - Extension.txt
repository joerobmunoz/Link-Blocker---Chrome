Link Blocker - Extension


Features:

CRUD Link
	- Name
	- Http link

Toggle block
	- Checkbox (on/off)

Set timer
	- Date time, then blocks access once timer is reached

Block until ...
	- Blocks link until set datetime has expired

Rules
	- Limit on site per day

Audit history?
	- Record total time/unit (day) of registered site access (when not blocked)
	- This could be useful for an experiment, to see the behavioral change in 
		users that participate in blocking (Behavioral mod., CBT, etc.)
	- Method to aggregate data
		- Option to send in anonymous usage data.
